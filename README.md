# Shayan Majidy - Academic Website

Personal academic website for Shayan Majidy, Banting Postdoctoral Fellow at Harvard University.

Built with Jekyll using a customized [academicpages](https://github.com/academicpages/academicpages.github.io) theme.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000` to preview.
